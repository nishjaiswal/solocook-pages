Before you email

"My streak disappeared."

Streaks count consecutive calendar days on which you tapped "I made this →" on a recipe. Missing a single day resets the streak to zero. The streak lives only on your device, so if you reinstalled the app or reset the onboarding, it starts fresh.

"Some recipe photos are missing or repeat."

We source recipe images from Unsplash. A handful of recipes in the catalogue don't have a dedicated image, so you'll occasionally see the same cuisine-level cover photo. The recipe itself is always its own recipe — only the picture is shared. The "A note on photos" card in the Results screen explains this.

"I selected 'Italian staples' and it also checked other ingredients."

Fixed in 0.1.0. If you're seeing this behaviour, update to the latest version from the App Store.

"How do I start over?"

Go to the You tab → scroll to Quietly dangerous → Reset everything. This wipes the name, the streak, the saved recipes, the money-saved counter, and returns you to the Welcome screen. There is no recovery; the app never had a copy of any of it.

"Can I sync between my iPhone and iPad?"

Not yet. SoloCook 0.1.0 stores all state on a single device. Cross- device sync is on the roadmap but requires user accounts, which we don't currently offer.

"Is there an Android version?"

Yes — SoloCook is also on Google Play. The two versions have feature parity; your data does not transfer between them, because everything lives on the device.
