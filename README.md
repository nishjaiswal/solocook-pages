# solocook-pages

Static pages for the SoloCook iOS + Android apps, hosted on GitHub Pages.

| Page | Purpose |
|---|---|
| `index.html` | Landing page linking to the other two |
| `privacy.html` | Privacy policy — linked from App Store + Play Store listings |
| `support.html` | Support / FAQ page — linked from App Store + Play Store listings |

## Updating

Edit the HTML files locally, commit, push. GitHub Pages rebuilds
automatically within ~30 seconds.

```sh
git add privacy.html
git commit -m "update privacy policy"
git push
```
